1. Awal Penemuan HTML (1989 - 1991)

HTML Hypertext Markup Language diciptakan oleh Tim Berners-Lee di CERN, Swiss.
Tujuannya sederhana: Untuk memudahkan ilmuwan berbagi dokumen penelitian secara digital.

  • 1991: Versi pertama (HTML 1.0) dirilis dengan hanya sekitar 18 tag dasar.
  • Karakteristik: Sangat sederhana, hanya teks dan link, tanpa gambar / desain yang rumit.

2. Perjalanan Versi HTML

Perkembangan HTML dikelola oleh W3C (World Wide Web Consortium) dan kemudian WHATWG (Web Hypertext Application Technology Working Group).

Versi Tahun Inovasi Utama
HTML 2.0 1995 Standar pertama yang mendukung formulir (forms)
HTML 3.2 1997 Dukungan untuk tabel, teks rata tengah, dan applet Java
HTML 4.01 1999 Pemisahan konten (HTML) dan desain (CSS)
XHTML 2000 Versi HTML yang lebih ketat berbasis XML
HTML5 2014 Dukungan multimedia, API JavaScript, dan elemen semantik baru

3. Era HTML 5: Standar Modern Saat Ini

HTML5 adalah versi terbaru dari HTML yang dirilis pada tahun 2014. Ia membawa banyak inovasi dan fitur baru yang membuat pengembangan web menjadi lebih mudah dan efisien.

  • Elemen Semantik: Pengenalan tag seperti <header>, <main>, dan <footer> untuk membantu mesin pencari (SEO) memahami struktur halaman.
  • Multimedia tanpa Plugin: Tidak lagi butuh Flash untuk memutar video / audio
  • Konektivitas: Mendukung penyimpanan lokal (local storage) dan aplikasi yang bisa berjalan offline

HTML 6: Evolusi Tulang Punggung Web

Sebagai fondasi web, evolusi HTML mencerminkan tuntutan teknologi dan kreativitas di era digital. HTML6 dikembangkan berdasarkan kerangka kerja yang kokoh dari versi sebelumnya, dengan memperkenalkan fitur-fitur inovatif yang dirancang untuk memenuhi kebutuhan aplikasi web yang semakin canggih.

Fitur Utama HTML 6:

  1. Elemen Semantic yang Ditingkatkan: Fitur ini bertujuan agar pengembang bisa membuat elemen kustom sendiri (seperti tombol atau menu khusus) dengan lebih mudah
    • Intinya: Mengurangi ketergantungan pada kode JavaScript yang rumit dan membuat struktur website lebih rapi dan mudah dikelola oleh tim pengembang.
  2. Interaktifitas Multimedia Bawaan: Untuk memutar video dengan kontrol canggih / menampilkan teks terjemahan, HTML 6 tidak perlu membutuhkan bantuan pihan ketiga karen dukungan langsung untuk:
    • Keterangan & Anotasi: Menambahkan catatan atau teks penjelasan langsung pada video/audio.
    • Kontrol Putar Lanjutan: Tombol kendali yang lebih lengkap dan interaktif tanpa perlu instalasi tambahan.
